# Node.js_practice

# Overview

This is a minimal Node.js project that creates a basic HTTP server without using any external frameworks like Express.
The server handles simple routing and serves static HTML files from the views directory.

# Features

Basic HTTP server using Node’s built‑in http module

Simple routing with a switch statement

Serves static HTML files (index.html, about.html, contact-me.html)

Custom 404 page

Logs every incoming request URL
